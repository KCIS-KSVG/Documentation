## 前言

本文件由昆山康桥学生摄影合作组织（KSVG）提出并统一归口。

本文件确立了采用本标准体系的实体组织在成员绩效量化管理方面的通用规则，是 *KS-5001 绩效监视与测量标准* 的具体执行附件。本文件通过建立标准化的绩效积分体系，旨在为成员的荣誉评定、岗位晋升及资源分配提供客观的量化依据，体现贡献导向与公平竞争的原则。

作为开源标准体系的一部分，本文件为各实体组织提供通用的绩效积分实施框架。本文件仅作为通用管理规范与框架提供，采用本标准的实体组织，本文件中所涉及的“责任”、“义务”、“分值评估”等表述，均指代组织内部的行政管理责任与自律约定，不自动构成法律层面的民事或刑事责任认定。采用方应自行釐清其所在司法管辖区内的法律边界。

本文本所指称的“组织”，泛指所有声明采用本开源标准体系运行的具体业务实体。当本文件条款与采用实体所在地的法律法规、上级监督机构（如母体机构、属地管理部门）发布的明文规章发生冲突时，须无条件以上位规章为准。

## 1 范围

本文件规定了成员绩效积分的获取途径、计算逻辑、扣除标准、结算周期及关联权益。

本文件适用于：
- a) 全体正式成员及考察期成员的日常业务表现记录；
- b) 业务周期末荣誉成员评选及岗位选拔的数据支撑。

## 3 术语和定义

下列术语和定义适用于本文件。

**3.1**
**基础分**
基于业务执行时长或任务量给予的固定基准积分，反映成员参与组织的活跃程度。

**3.2**
**质量分**
基于产出物的技术指标、艺术价值或品牌贡献等级给予的额外奖励积分。

**3.3**
**红线扣分**
因触犯核心管理红线或发生重大违规行为而实施的惩罚性减分。

## 4 积分获取规则

### 4.1 业务执行基础分
成员参与经 *KS-4001* 正式立项的项目，依据项目等级与实际核定工时计算积分。

| 项目/活动等级 | 计分标准 | 单次封顶分值 | 说明 |
| :--- | :--- | :--- | :--- |
| **重大项目 (S级)** | 2.0 分 / 小时 | 20 分 | 权重最高，通常涉及复杂环境与高强度执行。 |
| **重点项目 (A级)** | 1.5 分 / 小时 | 15 分 | 组织常规的核心业务活动。 |
| **常规任务 (B级)** | 1.0 分 / 小时 | 10 分 | 日常记录或基础性支持任务。 |
| **行政/技术例会** | 2.0 分 / 次 | - | 需全程参与并完成签到确认。 |

### 4.2 技术与质量奖励分
由执行机构或行政机构在产出物验收或阶段性评审后予以评定。

| 奖励细项 | 分值设定 | 评定基准 |
| :--- | :--- | :--- |
| **精选作品入选** | +5 分 / 件(组) | 作品入选组织官方年度作品集或被顶级合作方采纳。 |
| **技术创新实践** | +10 分 / 次 | 成功应用新技术流程并产生显著的质量提升效果。 |
| **知识库贡献** | +5 ~ 15 分 / 篇 | 撰写并提交被知识库采纳的技术教程或风险规避指南。 |
| **技术培训讲授** | +10 分 / 课时 | 担任内部技术培训讲师并有效完成课程交付。 |

### 4.3 组织协同奖励分
鼓励成员表现出超越岗位职责的协作精神。

| 奖励细项 | 分值设定 | 说明 |
| :--- | :--- | :--- |
| **紧急任务响应** | +5 分 / 次 | 在任务启动前极短时间内临时接受调遣并有效补位。 |
| **业务周期全勤** | +20 分 / 周期 | 周期内所有排班任务无任何迟到、缺席或请假记录。 |
| **资产维护贡献** | +2 分 / 次 | 主动发现并上报重大资产安全隐患，或协助完成资产盘点。 |

## 5 积分扣除规则

对违反 *KS-2003* 及其他受控管理规定的行为实施减分。

### 5.1 出勤违规扣分
| 违规细项 | 扣分标准 | 备注 |
| :--- | :--- | :--- |
| **轻微迟到 (≤15分钟)** | -2 分 | 同时扣除该次任务的基础分。 |
| **严重迟到 (>15分钟)** | -5 分 | 严重影响团队协作效率。 |
| **无故缺席** | -20 分 | 严重损害组织声誉与协作契约。 |
| **非预期的临时请假** | -5 分 | 任务开始前 24 小时内请假且未协助落实补位人员。 |

### 5.2 执行与管理违规扣分
| 违规细项 | 扣分标准 | 备注 |
| :--- | :--- | :--- |
| **数据备份违规** | -10 分 | 采集后未按 *KS-3002* 规定完成数据校验与备份。 |
| **资产归还违规** | -5 分 | 归还时未执行基础清洁或未重置设备状态。 |
| **职业形象不符** | -2 分 | 正式执行场合未按规定着装或未佩戴身份标识。 |
| **交付时效违约** | -5 分 / 日 | 未经授权批准而延迟交付成品。 |

### 5.3 熔断性扣分
若成员触犯 *KS-1003* 定义的任何安全或合规红线行为（如私自挪用资产、泄露敏感隐私、发生重大安全事故等），除**积分账户立即清零**外，行政机构将启动强制除名程序。

## 6 积分结算与管理流程

### 6.1 记录与申报
- a) **主动申报**：任务结束后 48 小时内，相关成员或项目负责人应在管理系统中提交工时与成果数据；
- b) **审核确认**：行政专员核对排班日志与交付记录，确认无误后录入绩效档案；
- c) **定期公示**：积分数据应每月在组织内部受控范围内公示，接受异议申诉。

### 6.2 积分有效期
绩效积分以一个完整的业务年度为累计周期。周期结束后，积分数据归档封存，作为荣誉评定的终值依据，下一周期重新计算。

## 7 积分关联权益

绩效积分作为衡量成员贡献度的核心指标，享有以下组织内权益优先权：

### 7.1 资源调度优先权
- a) **资产领用优先**：在同等能力认证等级下，积分较高者拥有热门或高价值资产的优先预约使用权；
- b) **机位岗位优选**：在大型协作项目中，积分较高者可优先选择核心技术机位。

### 7.2 晋升与评定准入
- a) **高阶认证准入**：申请 L3 级及以上技术认证考核，需满足一定的累计积分阈值；
- b) **管理岗位竞选**：竞选管理职务时，累计积分排名应处于组织前列；
- c) **荣誉称号评定**：优秀成员及技术贡献奖的评选，应严格基于积分排行进行差额选拔。
