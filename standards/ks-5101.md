## 前言

本文件由 **昆山康桥学生摄影合作组织 (KSVG)** 提出并归口。

本文件是 *KS-5001 绩效监视与测量标准* 的具体执行附件。本文件将抽象的“贡献度”转化为可计算的数字化指标（积分），为组织的评优、晋升及资源分配提供量化依据。本文件旨在建立一套“多劳多得、优劳优得、违规必究”的公正评价体系。

## 1 范围

本文件规定了积分的获取途径、计算公式、扣除标准、结算周期及兑换权益。

本文件适用于：
a) 全体正式成员及考察期成员的日常绩效记录；
b) 学期末优秀成员评选的数据支撑。

## 3 术语和定义

下列术语和定义适用于本文件。

**3.1**
**基础分 (Base Points)**
基于服务时长或任务数量给予的保底积分，反映成员的“苦劳”。

**3.2**
**质量分 (Quality Points)**
基于产出物的质量等级给予的额外奖励积分，反映成员的“功劳”。

**3.3**
**红线扣分 (Critical Deduction)**
触犯严重违规行为时的惩罚性扣分，通常伴随行政处分。

## 4 积分获取规则

### 4.1 任务基础分
成员参与经 *KS-4001* 立项的项目，依据项目等级与工时计算积分。

| 项目等级 | 计分标准 | 单次封顶 | 说明 |
| :--- | :--- | :--- | :--- |
| **S级 (重大)** | 2.0 分 / 小时 | 20 分 | 涉及校级大型活动，压力大，权重最高。 |
| **A级 (重点)** | 1.5 分 / 小时 | 15 分 | 常规重要活动。 |
| **B级 (常规)** | 1.0 分 / 小时 | 10 分 | 日常会议记录、简单扫街。 |
| **行政例会** | 2.0 分 / 次 | - | 需全程参与并签到。 |

### 4.2 质量奖励分
由执行机构或行政机构在审片/验收后评定。

| 奖励项 | 分值 | 评定标准 |
| :--- | :--- | :--- |
| **精选作品** | +5 分 / 张(组) | 作品入选组织官方精选集 (Showcase) 或被校方官方账号采用。 |
| **技术突破** | +10 分 / 次 | 首次成功应用新技术（如延时摄影、特效合成）并产出有效成果。 |
| **教程贡献** | +5 ~ 15 分 / 篇 | 撰写高质量技术教程或避坑指南并收录入知识库（视篇幅与深度而定）。 |
| **培训讲师** | +10 分 / 课时 | 担任内部培训讲师并完成授课。 |

### 4.3 行为奖励分
鼓励互助与奉献精神。

| 奖励项 | 分值 | 说明 |
| :--- | :--- | :--- |
| **救火队员** | +5 分 / 次 | 在任务开始前 4 小时内，临时替补突发缺席的成员。 |
| **全勤奖** | +20 分 / 学期 | 学期内所有排班无迟到、无请假、无早退。 |
| **资产维护** | +2 分 / 次 | 主动发现器材隐患并报告，或协助完成资产大盘点。 |

## 5 积分扣除规则

对违反 *KS-2003* 及其他管理规定的行为实施扣分。

### 5.1 考勤违规
| 违规项 | 扣分标准 | 备注 |
| :--- | :--- | :--- |
| **迟到 (≤15分钟)** | -2 分 | 扣除当次任务基础分。 |
| **迟到 (>15分钟)** | -5 分 | 视为严重迟到，可能影响任务执行。 |
| **无故缺席 (No-show)** | -20 分 | “鸽子”行为，严重影响团队声誉。 |
| **临时请假** | -5 分 | 任务开始前 24 小时内请假且未找到替补。 |

### 5.2 执行违规
| 违规项 | 扣分标准 | 备注 |
| :--- | :--- | :--- |
| **素材未备份** | -10 分 | 拍摄后未按 *KS-3002* 规定上传/备份原始素材。 |
| **器材未归位** | -5 分 | 归还时未清理镜头、未重置设置或未放入指定位置。 |
| **着装不整** | -2 分 | 正式场合未佩戴工作证或穿着拖鞋/短裤等。 |
| **延迟交付** | -5 分 / 天 | 未按 *KS-4005* 承诺的时间交付成品。 |

### 5.3 熔断扣分
触犯 *KS-1003* 规定的红线行为（如私用器材、泄露隐私、安全事故），除**积分清零**外，直接启动除名程序。

## 6 积分结算与管理

### 6.1 记录流程
1.  **申报**：任务结束后 48 小时内，项目负责人或成员本人在管理系统中提交工时与成果申报。
2.  **审核**：行政专员核对排班表与交付物，确认无误后录入系统。
3.  **公示**：积分榜单每月初在内部群组公示 3 天，接受异议申诉。

### 6.2 有效期
积分以**学年**为单位进行累计。每学年结束后（通常为暑假前），积分清算归档，下一学年重新起算。

## 7 积分权益

积分主要用于衡量贡献度，不与金钱挂钩，但享有以下组织内权益：

### 7.1 优先权
*   **器材借用**：在同等技能等级下，积分高者拥有热门器材（如最新相机）的优先预约权。
*   **机位选择**：在 S 级大型活动中，积分高者可优先选择核心机位（如舞台正中、游机位）。

### 7.2 晋升门槛
*   **晋升 L3**：申请 L3 级技能认证考核，需累计积分达到 50 分。
*   **竞选干部**：竞选部长级岗位，需累计积分达到 100 分且排名前 30%。

### 7.3 荣誉评定
*   **年度优秀摄影师**：学年积分总排名前 3 的成员。
*   **积极分子**：学年积分达到 80 分以上的成员。
